# Interactive Business Dashboard using Streamlit

## Objective
Develop an interactive business intelligence dashboard for analyzing sales and profit performance.

## Dataset
Business dashboard Dataset

## Approach

1. Data cleaning and preprocessing
2. Exploratory Data Analysis
3. KPI calculation
4. Streamlit dashboard development
5. Interactive filtering and visualization

## Key Features

- Region and category filters
- Total sales and profit metrics
- Top 5 customers by sales
- Sub-category sales visualization

## Tools Used

- Python
- Pandas
- Matplotlib
- Streamlit

## Results

The dashboard enables interactive analysis of business performance and helps identify top-performing products and customers.
